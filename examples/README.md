# Context Capsule Examples

This directory contains example implementations and use cases.

## Structure

- `basic/` - Basic usage examples
- `advanced/` - Advanced features and patterns
- `integrations/` - Integration examples with other tools

## Getting Started

Each example includes:
- Source code
- README with explanation
- Comments explaining concepts

Run examples:
```bash
cd examples/basic
npm install
node index.js
```

## Available Examples

### Basic
- Creating a context capsule
- Saving sessions
- Creating snapshots

### Advanced
- Multi-project management
- Context synchronization
- Custom context structures

### Integrations
- Git integration
- CI/CD pipeline integration
- IDE plugins
